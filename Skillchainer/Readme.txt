Addon Commands:

'sc help' - displays this help
'sc party #'(1-2) - sets party size
'sc p2 name' - sets the other party member\'s name for SC purposes
'sc member #'(1-2) - sets user to member 1 or 2 for SC purposes
'sc selfsc on|off' - turn self skillchaining on or off
'sc pull on|off' - turns auto pulling on or off if user member is set to 1
'sc run on|off' - turns auto running to engaged mob on or off
'sc provoke on|off' - turns provoking on or off if warrior is main or sub
'sc assist "name"' - sets a person as the assist, no quotes required
'sc pause on|off' - pauses the skillchainer
'skillchainer pause

'sc addws "Weaponskill" Step' | 'removews Step' - adds or removes the self skillchain weaponskill
example: //sc addws "Tachi: Fudo" 1 and //sc removews 1

'sc listws' - lists all the self skillchain steps and weaponskills for current job
'sc ws on|off' - turns the use of weaponskills on or off
'sc mainws|subws "Weaponskill"' - changes the weaponskills used when either solo spamming or skillchaining with another player
'sc addmob|removemob mob' - adds or removes mob to the target auto pull list
